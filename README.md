
# Mintos

Mintos is a file asset manager allowing users to upload files they wish to keep track of and mint those file assets to the ethereum blockchain.

<br>

## Technologies Used 

- Javascript
- HTML
- CSS
- MERN
- Bootstrap
- Solidity
- Metamask
- Alchemy
- Pinata
